# Python学生信息管理系统 python361

## 项目概述

本项目是基于Python的Flask框架开发的一款学生信息管理系统，旨在帮助学校或教育机构高效地管理学生信息、成绩、班级、课程、宿舍等。

## 技术栈

- **开发语言：** Python
- **框架：** Flask
- **前端技术：** HTML、CSS、JavaScript、jQuery
- **数据库：** MySQL
- **工具：** Vscode、Navicat
- **运行环境：** Python 3.0以上、MySQL 8.0

## 功能模块

- **用户登录与注册**
- **学生管理：** 添加、编辑、查询、删除学生信息
- **成绩管理：** 成绩录入、修改、查询
- **班级管理：** 班级信息维护
- **课程管理：** 课程信息维护
- **宿舍管理：** 宿舍分配与管理
- **任务管理：** 发布、跟踪学生任务
- **资料管理：** 学生资料的上传与下载
- **意见反馈：** 学生或教师提出意见与建议
- **系统管理：** 系统设置与维护

## 系统角色

- 系统管理员
- 教师
- 学生

## 运行环境

- Python 3.0及以上版本
- MySQL 8.0数据库

## 部署步骤

1. 安装Python环境及MySQL数据库
2. 克隆或下载项目代码
3. 安装项目依赖
4. 配置数据库连接信息
5. 运行项目
